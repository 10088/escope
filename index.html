Escope (escope) is ECMAScript scope analyzer extracted from esmangle project.

Document

Generated JSDoc is here.

Demos and Tools

Demonstration is here by Sasha Mazurov (twitter: @mazurov). issue

And there are tools constructed on Escope.

  • Esmangle is a minifier / mangler / optimizer.
  • Eslevels is a scope levels analyzer and SublimeText plugin for scope context coloring is constructed on it.
  • Esgoggles is JavaScript code browser.
